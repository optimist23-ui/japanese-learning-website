JAPANESE LESSONS MOBILE APP

This is an installable web app (PWA) containing the complete HTML course and its audio.

HOW TO SHARE BY TEXT MESSAGE
1. Upload this folder to a secure HTTPS web host.
2. Text the web address to students or instructors.
3. The recipient opens the link in Safari or Chrome.

HOW TO INSTALL ON IPHONE OR IPAD
1. Open the web link in Safari.
2. Tap Share.
3. Tap Add to Home Screen.

HOW TO INSTALL ON ANDROID
1. Open the web link in Chrome.
2. Tap Install App, or open the browser menu and choose Add to Home Screen.

IMPORTANT
- The app cannot be installed from a file:// address.
- Installation and offline caching require HTTPS hosting.
- Lessons and audio are cached as they are opened, which avoids a very large first download.
- Editing changes remain saved in the browser where they were made.
- Review-card Japanese text is sized to fit the card without overpowering the image.
- Romaji is separated into Japanese pronunciation units, for example:
  o · ha · yo · u   go · za · i · ma · su
- The 日本語 / English button changes the full teaching interface for Japanese-speaking users.
- The selected interface language is remembered on that device.
