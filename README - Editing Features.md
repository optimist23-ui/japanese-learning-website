## Utilize All editor

The `Utilize All` button provides PowerPoint-style course editing:

- Upload one or more image slides.
- Duplicate or delete a slide.
- Move slides earlier or later inside their lesson.
- Edit titles, notes, video links, and replacement images.
- Restore the most recently deleted slide.
- Export all course edits and uploaded images to JSON.
- Import a previously exported course-edit file.
- Restore the complete original course.

Edits are stored locally in the browser. Use Export Course Edits to transfer or
back up changes.

## Romaji phonetic support

- Lesson-navigation Japanese subtitles include romaji.
- Vocabulary and sentence reveal cards show romaji beneath the Japanese.
- Audio guessing-card answers show romaji beneath the Japanese sentence.
- Romaji is presented as learner-friendly phonetic spelling while Japanese
  remains the primary instructional text.
