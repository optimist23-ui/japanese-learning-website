# Japanese Lessons and Cultural Exchange

Interactive bilingual Japanese teaching course containing:

- Japanese language lessons 1–9
- Cultural Exchange modules grouped with their related lessons
- Native Japanese lesson audio
- Anime decoding review cards
- Audio guessing cards
- English and Japanese instruction videos
- “All Slides” lesson navigation
- “Utilize All” slide editing, upload, deletion, reordering, import, and export tools
- Installable mobile web app support

Open `index.html` or use the GitHub Pages site.
